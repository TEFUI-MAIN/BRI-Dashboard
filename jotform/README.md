# Kartar Jotform Operations Dashboard

Static GitHub Pages dashboard for weekly Jotform operations reporting.

## Current dashboard

- Runsheet PDF and attachment completion KPIs
- Pre-start and fit-for-duty compliance KPIs
- Weekly and monthly trends
- Driver productivity and compliance table
- Vehicle maintenance queue by rego
- Missing pre-start queue
- Site workload summary

## Data note

This first version is a static dashboard snapshot based on recent Jotform data available during build. A future version can call a backend/API export job to refresh the data automatically.
